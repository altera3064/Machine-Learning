- Feature(특성)

데이터를 표현하는 변수를 의미한다. 

예를 들어, 

공부 시간 -> Feature (입력)
시험 점수 -> Target (출력)

- Weight(가중치)

< 가중치의 크기와 모델의 의미 >

가중치는 모델이 어떤 특성이 더 중요한지를 결정하는 역할을 합니다. 쉽게 말해, 가중치가 클수록 해당 특성이 결과에 미치는
영향력이 크다고 모델이 판단한다. 

가중치가 커지는 경우 

- 특정 특성에 대한 의존도가 매우 높아짐
- 작은 변화에도 모델 출력이 민감하게 반응 -> 조그마한 값의 변화에도 값이 커지니까
- 과적합할 확율이 높아진다. 

- Bias(절편)
출력값의 기준점 역할을 한다. 즉, 입력값이 모두 0일때 모델이 예측하는 출력값을 나타낸다.

*절편 예제 이미지를 확인

- Coefficient(계수)

회귀 모델에서는 가중치는 = 계수라고 말할 수 있다.
